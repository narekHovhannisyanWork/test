# Wildberries Tariffs Data Service

## Описание

Сервис для получения тарифов с API Wildberries, сохранения данных в PostgreSQL и выгрузки их в несколько Google Sheets.

## Требования

- Node.js >= 16
- Docker
- PostgreSQL
- Google Sheets API
